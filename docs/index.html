WeRead Vault

微信读书的本地数据库，一个带 Agent Skill 的 CLI。

把书架、划线、想法、阅读统计归档进本机 SQLite —— 可查询、可视化、可导出，AI 直接读，数据不上传。

下载 macOS 下载 Windows GitHub ↗

个人数据归档工具，不是官方客户端，只同步你自己的数据。

安装与使用

下载即用，无需 git clone。

下载 macOS (.dmg) 下载 Windows (.exe)
三步用起来：
1. 打开：macOS 打开 dmg 把 App 拖进「应用程序」双击；Windows 双击 exe。
   首次系统会提示未签名 —— macOS 右键 App →「打开」，Windows 点「更多信息 → 仍要运行」（一次即可）。
2. 同步：启动后浏览器自动打开，在「同步设置」粘贴微信读书 API Key（从这里获取），点「同步」。
3. 用：浏览书架、看统计、搜索、导出到 Markdown / Obsidian / Notion。

习惯命令行？一行启动（需 Python 3.10+）：

uvx weread-vault serve --open
或者，把这段发给 AI agent 让它帮你装 ▾ 
请用 pipx 安装并运行 WeRead Vault：
pipx install weread-vault && weread-vault serve --open

本地优先

完整书架
